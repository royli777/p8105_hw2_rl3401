Import, clean, and tidy the dataset of baseline demographics

``` r
baseline_df = 
read_csv("data/MCI_baseline.csv", skip = 1,show_col_types = FALSE) |>
  janitor::clean_names()  |>
  filter(age_at_onset == "." | age_at_onset > current_age)|>
   mutate(
      sex = case_match(
      sex,
      1 ~ "male",
      0 ~ "female"),
      apoe4  = case_match(
      apoe4,
      1 ~ "carrier",
      0 ~ "non_carrier"),
      current_age = as.numeric(current_age)
  ) 
```
